## RustSiris | A Simple Alkad Rust Cheat
### A C++ Cheat for the pirate version of Rust (Alkad Rust)
### Features:
* ESP:
*    Names
*    Distance
*    Skeleton
*    Player Boxes (Box, Corner Box)
*    Snaplines
*    Show Sleepers
*    Show Wounded
*    OOF Arrows (OOF Arrows Distance
* Aimbot:
*    Aimbot KeyBind
*    Smoothing
*    Bone Selection
*    Fov
*    Show Fov
*    Show Aimbot Target [BETA]
* Self:
*    Fake Admin
